# ROS2CRAFT manager package

Main ros2craft ros2 package to interface with MCrobots. Main class is MCRobot LINK, loaded and unloaded through ros2 component system as a dynamic node by calling the respective services.
For now just he launch file LAUNCH FILE is needed, and the robots will be addded at command from MC.
